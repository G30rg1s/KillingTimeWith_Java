Φτίαξτε μια κλάση Book που θα διαθέτει τα ακόλουθα χαρακτηριστικά:
Μεταβλητή title: Ένα string για την αποθήκευση του τίτλου του βιβλίου.
Μεταβλητή  isAvailable: Μία boolean εάν είναι διαθέσιμο ή οχι για δανεισμό
Constructor που δέχεται την ως όρισμα και τον τίτλο και θα αρχικοποιεί την isAvailable σε true
Μέθοδος printAvailability που τυπώνει τον τίτλο + "Διαθέσιμο για ενοικίαση" αν είναι διαθέσιμο και "Μη διαθέσιμο για ενοικίαση" αν όχι.

Φτίαξτε μια κλάση Member που θα διαθέτει τα ακόλουθα χαρακτηριστικά:
Μεταβλητή name: Ένα string για την αποθήκευση του ονόματος.
Μεταβλητή surname: Ένα string για την αποθήκευση του επώνυμου.
Μεταβλητή book: τύπου Book που θα αποθηκεύει το βιβλίο στο χρήστη.
Constructor που δέχεται την ως όρισμα και το όνομα και το επώνυμο και θα αρχικοποιεί το book σε null.

Μέθοδος addBook που θα παίρνει ώς παράμετρο ενα Book και ελέγχει αν είναι διαθέσιμο το βιβλίο και αν το μέλος δεν εχει βιβλίο. Αν ναι και στα δυο αποθηκέυει το βιβλιο στο μέλος και θετει την isAvailable σε false. Aν όχι δεν αποθηκέυει τιποτα.

Μέθοδος returnBook που ελέγχει το μέλος έχει βιβλίο. Αν έχει, θετει την isAvailable σε true και θέτει το book σε null. Aν όχι δεν αλλάζει τιποτα.

Μέθοδος getMemberInformation που ελέγχει αν o χρήστης έχει δανειστέι βιβλίο και επιστρέφει το όνομα του μέλους + τον τίτλο του βιβλόυ. Αν οχι το επιστρέφει το όνομα του μέλους + δεν έχει δανειστεί βιβλίο

Φτιάξτε δυο νεά βιβλία με τίτλους "Java A Beginner’s Guide" & "Java for Dummies"

Να νοικιάσει η Ann το πρωτο βιβλιο και τυπώστε τι έχει ενοικιάσει η Ann
